Behind the build

A real product, not a mockup

Every feature here is fully functional — real authentication, a real relational database, and real playback. A few decisions worth knowing about:

🔐

Stateless authentication

bcrypt-hashed passwords and JWT tokens verified per-request via a custom decorator — no server-side sessions to manage.

🎧

Real playback, no media hosting

Playback runs on the YouTube IFrame Player API, fully wrapped by a custom UI — no visible embed, no licensing or storage overhead.

🗄️

Relational data model

PostgreSQL schema with proper foreign keys across users, songs, playlists, likes, and follows — not a flat document store.

🎨

Hand-built design system

No UI kit — typographic album art, a custom animated equalizer motif, and a colour system built from scratch.

Full technical write-up in the GitHub README.
